# Selenium Google Search Testing Project

## Overview
This project demonstrates automated testing of Google search functionality using Selenium WebDriver.

## Features
- Search for a keyword in Google.
- Validate search results.
- Check response times for search queries.

## Prerequisites
- Java 11 or higher
- Maven
- Selenium WebDriver
